# ColorVisorWin
WPF app able to name color under cursor. Mainly for colorblind people. 
Uses dE2000 to calculate  the best matching name. 
Requires data.csv file with data in exe folder.

## data.csv format  
<color name\>,#<hexvalue\>  
every data entry must be in new line  
### Example  
Royal blue,#1232FF  
Red,#FF0000
